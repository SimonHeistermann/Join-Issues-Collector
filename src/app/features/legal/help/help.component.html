Welcome to the help page for Join. Here, you will find an overview of our kanban project management tool, its benefits, and how to use it effectively.

1. What is Join?

Join is a kanban-based project management tool developed as part of a web development bootcamp at the Developer Akademie.

Kanban, a Japanese term meaning "billboard", is a method that helps visualize work, limit work-in-progress, and maximize efficiency. Join follows these principles to provide an intuitive and visual interface for task and project management.

Please note that Join is an educational project and is not intended for professional business use. We do our best to ensure usability but cannot guarantee availability, accuracy, or reliability.

2. How to use Join

Below is a step-by-step guide on how to use Join effectively.

2.1 Exploring the Board

After logging in, you will see the default board, which represents your project. It consists of four lists: "To Do", "In Progress", "Await Feedback", and "Done".

2.2 Creating Contacts

You can add contacts to collaborate on projects. Navigate to the "Contacts" section, click "New Contact", and enter the required details. Contacts can be assigned tasks and interact with them on the board.

2.3 Adding Cards

Tasks are represented by cards. Click the "+" button under a list to create a new card. You can add a task name, description, due date, and assignees.

2.4 Moving Cards

As a task progresses, move it across lists by dragging and dropping the card to the appropriate stage.

2.5 Deleting Cards

Completed tasks can either be moved to the "Done" list or deleted. Deleting a card removes it permanently. Please proceed with caution, as this action is irreversible.
